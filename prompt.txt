─────────────────────────────
# 입력
- 페이지 목록: ["동물 목록"] 

─────────────────────────────
# 조건 / 제약
1. 모든 프롬프트는 **데스크탑 웹 전용** (모바일/반응형 제외)
2. 페이지별로 UI 요소를 빠짐없이 나열
3. 민감 정보는 보호소/관리자 권한 기준으로 표시
4. 색상, 폰트, 카드 크기, 마진·패딩 등 **프로젝트 CSS 규약 준수**
   - 색상 관련 세부 내용은 **styles/common.css 파일 참조**: 지정 CSS 변수만 사용
5. UI 요소 포함: 카드, 리스트, 버튼, 배너, CTA, 헤더, 푸터, 사이드바 등
6. 화면 배치, 위치, 크기 명시 (폭, 높이, 마진, 패딩 등)
7. 상호작용 및 상태 명시: 호버, 클릭, 모달, 팝업, 로딩, 빈 상태, 에러
8. 행동 유도: 사용자가 취할 행동과 UI 목적
9. 접근성 필수 적용: aria-label, role, 키보드 내비게이션, 이미지 alt
